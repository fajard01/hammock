Developer's Notes

A personal project developed with HTML, CSS (SCSS) and Javascript.

Hosted in GitHub, GitHub's site generator Jekyll generates the site, and utilizes Bootstrap framework for responsiveness, and Lightbox JS for automatic image handling. JSON and YAML files handle the data.

With close to 5oo images to load, the site is slow so please be patient for it to load. Optimization is something I'm still working on. Also I implemented certain elements that don't bode well for mobile devices, so it is recommended to view the site on a laptop or desktop. Tablets might just work.

Anyway, enjoy the site!
