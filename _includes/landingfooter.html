Developer's Notes

A personal project developed with HTML, CSS (SCSS) and Javascript.

Hosted in GitHub the site is generated by GitHub's internal site generator Jekyll, and uses Bootstrap framework for responsiveness, and Lightbox JS for automatic image handling. Json and YAML files manage data.

With close to 5oo images to load, the site is a bit slow so please be patient. Optimization is something I'm still working on. Also I use certain elements that don't bode well for mobile devices so it is best to view the site on a laptop or desktop. Tablets might just probably work.

Anyway, enjoy the site!
